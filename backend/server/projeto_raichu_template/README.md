# Projeto gerado - RaichuWebServer

Docs rápidos:
- Instalar: ./install.sh
- Rodar dev: ./run_dev.sh
- Criar controller: ./cli.py create-controller Nome
- Criar model: ./cli.py create-model Nome

Arquivos:
- app/templates/index.html -> versão Flask + Tailwind
- frontend/index.astro -> versão Astro lendo assets/apresentacao.md
- assets/apresentacao.md -> fonte única
