# Projetos Backend Pedro Victor 2026

---

5 servidores backend em backend para que fiquem
  prontos para o seu currículo e repositório.

  Aqui está o que criei para cada microserviço:

  1. ExpressServerNodeJS:
      • package.json - Configuração de dependências
      Express e Cors.
      • server.js - Servidor Express funcional com
      endpoint de status.
  2. Raichu-Web-Server-fastapi:
      • requirements.txt - Dependências FastAPI e Uvicorn.
      • main.py - API FastAPI com Schemas Pydantic
      simulando tarefas de estudos de Engenharia.
  3. Pikachu-API-Rest:
      • requirements.txt - Flask e Flask-SQLAlchemy.
      • app.py - Flask API integrada a banco SQLite
      gerando tabelas e rotas dinâmicas.
  4. Charizard-Drogon:
      • CMakeLists.txt - Arquivo de build cmake vinculando
      a biblioteca Drogon.
      • main.cc - Ponto de entrada C++ estruturando
      o servidor Drogon C++.
  5. Rayquaza Web-Server Cloud:
      • serverless.yml - Configuração para deploy
      serverless via AWS Lambda.
      • handler.py - Script FastAPI usando Mangum para
      empacotar a aplicação em ambiente serverless.
